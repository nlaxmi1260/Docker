Docker File for static html web site

  1. To create a container with reqired Software with your artifacts
  2. FROM: represents base
  3. WORKDIR: is the location to place your artifacts
  4. COPY: copy your artifacts to WORKDIR
  5. EXPOSE: expose required port
  6. CMD: execute required command
  7. # Use the official Nginx base image from Docker Hub
    FROM nginx:alpine

    # Set the working directory in the container
    WORKDIR /usr/share/nginx/html

    # Copy the local HTML files to the Nginx default directory
    COPY . .

    # Expose port 80 to the outside world
    EXPOSE 80

    # Start Nginx
    CMD ["nginx", "-g", "daemon off;"]

Docker Commands

  1. docker build -t [imageName] . : Command to create image from docker file . represents dockerfile present in current directory
  2. docker run -d -p 8080:80 [imageName] : Command to create container and run, -d run in detach mode, -p to map local port to container exposed port, 8080 is the local port and 80 is the container exposed port
